Welcome to the Chicago InfoSec Events Calendar FAQ!

The purpose of this effort is to coordinate scheduling for all the in-person security events in the Chicagoland area.

There are multiple ways to access the calendar:

Public Calendar:
https://calendar.google.com/calendar/embed?src=chicagoinfosecevents@gmail.com&ctz=America%2FChicago

Public iCal:
https://calendar.google.com/calendar/ical/chicagoinfosecevents@gmail.com/public/basic.ics

To integrate with your personal Google calendar:

Posting events:
If you would like to promote events, you can send a calendar invite ChicagoInfoSecEvents@gmail.com. FYI: The sending email will be visible on the public calendar.
Note: If you frequently host events (i.e. ISACA, BurbSec, ISSA, CSA, etc.), the calendar organizers will reach out and ask if you want calendar admin access to manage your events. Direct access will be limited to people well-known by one of the organizers, since they could delete events created by other groups.

Events should include the following:
- Event organizer
- Event name
- Date/time
- Location
- URL
  * If the event is NOT open to everyone, please state by invitation (or members) only, and do NOT include a URL.
- Please include the direct URL, and NOT a shortened URL (i.e. no bit.ly, goo.gl, etc. links)
- Please don't include attachments

Only events that meet the following conditions should be posted:
- Must be in person
- Local to Chicago or hosted by an active member of the Chicago community
  * Exceptions will be made for big events that pull numerous attendees from the Chicago area such as RSA, BlackHat, etc.
- Discount codes appreciated, but not required
